# Project Report - HTML 폼 프로젝트 결과물

## 📋 프로젝트 개요
부트캠프 3일차 학습을 바탕으로 제작한 실제 웹 폼 프로젝트들입니다. 학습한 HTML 폼 요소들을 활용하여 실용적인 웹 페이지를 구현했습니다.

## 🎯 완성된 프로젝트

### 1. 설문조사 양식 (설문조사 양식.html)
**목적**: 사용자 정보 수집 및 의견 조사를 위한 종합적인 설문 폼

**주요 기능**:
- **개인정보 섹션**
  - 이름, 나이, 성별, 이메일, 거주지 입력
  - 숫자 입력 범위 제한 (나이: 1-120)
  - 이메일 유효성 검증
- **설문 문항**
  - 체크박스를 이용한 관심사 다중 선택
  - 스포츠, 음악, 영화 등 선택 옵션
- **기타 의견**
  - 자유 텍스트 입력 영역 (50x5 크기)
- **네비게이션**
  - 페이지 내 앵커 링크로 섹션 간 이동

**기술적 특징**:
- 시맨틱 HTML 구조 (`<header>`, `<main>`, `<footer>`)
- `<fieldset>`과 `<legend>`로 논리적 그룹화
- 접근성을 고려한 `<label>` 연결
- 페이지 내 네비게이션 구현

### 2. 회원가입 (회원가입.html)
**목적**: 웹사이트 회원가입을 위한 사용자 정보 입력 폼

**주요 기능**:
- **필수 입력 사항**
  - 이름, 아이디, 비밀번호, 이메일, 휴대폰 번호
  - 아이디 입력 가이드 (6자 이상 20자 미만)
  - 비밀번호 보안 입력 (대소문자 구분)
  - 전화번호 전용 입력 타입 (`tel`)
- **선택 입력 사항**
  - 취미 다중 선택 (체크박스)
  - 성별 단일 선택 (라디오 버튼)
- **액션 버튼**
  - 가입하기 (submit)
  - 취소 (button)

**기술적 특징**:
- 다양한 input 타입 활용 (`text`, `password`, `email`, `tel`)
- `placeholder`를 통한 사용자 가이드
- 라디오 버튼과 체크박스의 적절한 활용
- 구조화된 리스트 형태의 폼 레이아웃

## 🔍 구현된 HTML 기술 요소

### 폼 구조
```html
<fieldset>
  <legend>섹션 제목</legend>
  <!-- 관련 입력 요소들 -->
</fieldset>
```

### 접근성 구현
```html
<label for="input-id">라벨명</label>
<input id="input-id" name="field-name" type="text">
```

### 다양한 입력 타입
- `text`: 일반 텍스트
- `password`: 비밀번호
- `email`: 이메일 주소
- `tel`: 전화번호
- `number`: 숫자 (범위 제한)
- `checkbox`: 다중 선택
- `radio`: 단일 선택

### 메타데이터 관리
```html
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="author" content="이현규">
```
