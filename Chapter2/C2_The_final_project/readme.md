## 🎯 C2_The_final_project (최종 프로젝트)

### 프로젝트 소개
Chapter 2에서 학습한 HTML 기초 지식을 활용하여 제작한 **개인 소개 페이지**입니다.

### 📋 프로젝트 상세 정보
- **제목**: 현규의 소개시간
- **개발자**: 이현규
- **목적**: HTML 기초 학습 내용 종합 실습

### 🛠️ 사용된 기술 및 태그
- **기본 구조**: `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`
- **텍스트 요소**: `<h1>`, `<h2>`, `<h3>`, `<p>`, `<strong>`, `<mark>`, `<em>`
- **이미지**: `<img>` 태그로 프로필 사진 삽입
- **목록**: `<ul>`, `<li>` 태그로 좋아하는 음식 리스트 작성
- **구분선**: `<hr>` 태그로 섹션 구분
- **메타데이터**: `charset`, `viewport` 설정

### 📝 페이지 구성 내용
1. **개인 소개** - 프론트엔드 개발자로서의 소개
2. **취미** - 개인적인 관심사와 취미 활동
3. **좋아하는 음식** - 선호하는 음식들을 목록으로 정리
4. **전공** - 컴퓨터소프트웨어학 전공 배경
5. **개발 동기** - 프론트엔드 개발을 시작하게 된 계기와 목표
