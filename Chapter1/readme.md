## 📚 Chapter 1: HTML 기초 공부

이번 챕터에서는 웹 페이지의 가장 기본이 되는 HTML의 구조와 문법을 학습했습니다.

### 1. HTML 문서 기본 구조 (`05-HTML문서기본구조.html`)
HTML 문서는 항상 표준 형식을 따라야 하며, 웹 브라우저가 문서를 올바르게 해석할 수 있도록 선언이 필요합니다.
* `<!DOCTYPE html>`: HTML5 문서임을 선언
* `<html>`: 전체 문서의 시작과 끝
* `<head>`: 문서의 메타데이터(제목, 인코딩 설정 등)를 포함
* `<body>`: 실제 사용자에게 보여지는 본문 내용

### 2. 기본 문법 및 태그 (`04-HTML기본문법.html`)
* **태그(Tag)**: 요소의 시작과 끝을 나타냄
* **속성(Attribute)**: 태그에 추가적인 정보나 스타일을 부여 (예: `id`, `style`)
    * `<p style="color: red;" id="text">`: 텍스트 색상을 빨간색으로 지정하고 고유 아이디 부여
* **주석(Comment)**: 코드 내 설명 작성 시 사용 (``)
* **기타 태그**:
    * `<p>`: 단락(Paragraph) 구분
    * `<hr>`: 수평선 삽입
