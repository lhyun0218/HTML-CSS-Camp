# 부트캠프 3일차 - CSS 기초

## 📚 학습 내용

### 1. CSS 문서 사용하기 (19번)
- **외부 스타일시트**: `<link>` 태그로 외부 CSS 파일 연결
- **내부 스타일시트**: `<style>` 태그 내부에 CSS 작성
- **인라인 스타일**: HTML 태그의 `style` 속성에 직접 작성

**CSS 적용 우선순위**
1. 인라인 스타일 (가장 높은 우선순위)
2. 나중에 작성된 스타일 규칙

### 2. 선택자 기본편 (20번)
**선택자 우선순위**
1. 아이디 선택자 (`#id`)
2. 클래스 선택자 (`.class`)
3. 태그 선택자 (`tag`)

**선택자 종류**
- 전체 선택자: `*`
- 태그 선택자: `h2`, `p`, `li`
- 클래스 선택자: `.hiphop`
- 아이디 선택자: `#belly`, `#poppin`
- 그룹 선택자: `h2, p, li`

### 3. 박스 모델 이해 (21번)
- **박스 모델 기본 개념** 학습
- **백분율(%) 단위**: 부모 요소를 기준으로 계산
- `width: 50%`, `height: 50%` 사용법

### 4. 테두리 스타일 (22번)
**테두리 속성**
- `border-width`: 테두리 두께
- `border-style`: 테두리 스타일 (solid, dashed 등)
- `border-color`: 테두리 색상
- `border`: 단축 속성 (두께 스타일 색상)

**테두리 둥글게**
- `border-radius`: 모서리 둥글게 처리
- 시계 방향으로 상좌 → 상우 → 하우 → 하좌 순서

### 5. 여백 다루기 (23번)
**여백 속성**
- `margin`: 외부 여백 (요소 간 간격)
- `padding`: 내부 여백 (콘텐츠와 테두리 간 간격)
- 방향별 설정: `top`, `right`, `bottom`, `left`

**박스 사이징**
- `box-sizing: border-box`: 테두리와 패딩을 포함한 전체 크기 고정

## 🎯 핵심 포인트

1. **CSS 적용 방법 3가지**를 이해하고 우선순위 파악
2. **선택자 우선순위**를 정확히 알고 활용
3. **박스 모델**의 기본 구조 이해
4. **테두리와 여백** 속성을 활용한 레이아웃 조정
